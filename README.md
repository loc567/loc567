## 项目简介
loc567 是一款**完全开源免费**的纯网页端iOS模拟定位工具。
无需电脑、无需越狱、无需巨魔(TrollStore)环境，兼容iOS12-iOS26全版本系统，无任何系统版本限制，支持长时间稳定模拟定位，可绕过钉钉、企业微信常规定位安全检测。

## 核心特性
1. 脱离电脑：仅iPhone原生Safari浏览器访问，无需数据线、无需电脑配合
2. 全系统兼容：覆盖iOS12至iOS26所有正式版本，无需巨魔、企业证书、越狱
3. 长效稳定定位：网页切后台依旧保活，定位不会自动掉线、失效
4. 低风控隐蔽性：无系统注入、无证书篡改，设备不会留存异常系统日志
5. 永久开源免费：无广告、无内购，个人非商用场景永久免费使用
6. 零安装成本：无需安装APP、描述文件、插件，打开网页直接使用

## 使用步骤
1. 使用iPhone自带Safari浏览器访问官网
2. 授予浏览器地理位置权限（所有数据仅设备本地运算，不上传云端服务器）
3. 搜索/手动选取目标坐标，一键开启模拟定位
4. 退出网页至后台，模拟定位状态持续生效

## 访问注意事项
结合实测报错：直接访问会提示**网页解析失败**，解决方式：
1. 仅支持iPhone原生Safari浏览器，Chrome、夸克等第三方浏览器无法适配
2. 需要合规网络环境才可正常访问，否则固定弹出解析失败提示
